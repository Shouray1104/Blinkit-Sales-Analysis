# Blinkit-Sales-Analysis
Cleaned and aggregated 1.2M+ sales records with SQL for product, outlet, and rating insights.  Conducted Python EDA revealing Regular (425K+) vs. Low-Fat (776K+) sales and outlet size effects.  Built Power BI dashboard showing total sales, avg rating 3.9, and top categories (Snacks &amp; Fruits: 180K+).
